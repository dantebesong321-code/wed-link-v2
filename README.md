# WedLink

WedLink is a wedding vendor discovery platform that helps couples find and connect with trusted wedding professionals in one place.

## Problem

Planning a wedding often requires searching through multiple websites, social media accounts, and directories to find photographers, venues, caterers, makeup artists, and other vendors. This process is time-consuming and fragmented.

## Solution

WedLink centralizes wedding vendor discovery into a single platform where users can:

- Browse wedding vendors by category
- Search vendors by name, city, or description
- View vendor profiles and details
- Explore wedding inspiration
- Add and manage vendor listings

## Features

### Users

- Browse all vendors
- Search vendors
- Filter vendors by category
- View vendor details
- Explore wedding inspiration

### Vendors

- Create a vendor profile
- Edit vendor information
- Upload business information and images
- Showcase services and contact details
